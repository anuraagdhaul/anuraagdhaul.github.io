Scratch

I first started learning to code by operating Scratch. Here, I built projects while mastering the language's different aspects: motion, looks, sounds, events, controls, sensors, and variables. Over the years, Scratch helped me learn to code, solve challenging problems, and approach situations logically and methodically. Learning Scratch not only helped me bring my imagination to life but also aided me when solving real-life problems.

This project is an interactive 2 versus 2 soccer game. Two teams with two players each are controlled by four people using unique controls:

Ben: Arrow Keys (Up, Down, Left, Right)

Hannah: WASD (W - Up, S - Down, A - Left, D - Right)

Avery: IJKL (I - Up, K - Down, J - Left, L - Right)

Jordyn: TFGH (T - Up, G - Down, F - Left, H - Right)

Python

After mastering Scratch, I started to learn how to code in Python. I spent a total of two years learning Python with an online coding instructor and created many projects and games. To this day, Python remains one of my favorite coding languages as it is simple to use and similar to English. The following project is my favorite game that I coded.

The first master project I created was a remake of the Squid Game. In this game, one controls the green triangle and aims to reach the multicolored square before the red circles touch the triangle. If the green triangle is touched by the red circles, they lose one life. I hope you enjoy playing this game.

To play the game use your arrow keys: (Up, Down, Left, Right)
